🔓 Rouvrir un match clôturé

Remet un match en in_progress et réinitialise le dernier set (supprimé s'il est vierge, sinon dé-finalisé). Le match réapparaît dans la bannière "EN COURS" sur index.html.

1. Connexion admin

⚠️ Non connecté

2. Sélection du match

Tous les matchs sont listés (Firebase + localStorage), quel que soit leur statut. Source indiquée à droite de chaque match.

3. Appliquer

Écrit dans Firebase et localStorage. Recharge index.html après pour voir la bannière.
